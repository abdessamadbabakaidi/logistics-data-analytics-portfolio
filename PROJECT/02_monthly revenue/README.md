# 02 - Monthly Revenue

## Project Overview
This project analyzes monthly freight revenue using SQL and Tableau to identify revenue trends over time.

## Objectives
- Calculate total monthly revenue
- Identify high and low revenue months
- Visualize revenue trends

## Tools Used
- SQL Server
- Tableau
- Excel

## Files Included
- SQL queries
- Tableau dashboard
- Dataset
- Project screenshots

## Key Insights
- Revenue fluctuates across different months
- Some months generated significantly higher freight revenue
- Tableau dashboard highlights revenue trends visually

## Skills Demonstrated
- SUM()
- GROUP BY
- ORDER BY
- Date analysis
- Data visualization
