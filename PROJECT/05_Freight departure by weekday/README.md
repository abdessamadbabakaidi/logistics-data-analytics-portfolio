# 05 - Freight Departure By Weekday

## Project Overview
This project analyzes freight departures by weekday using SQL and Tableau to identify shipping activity patterns throughout the week.

## Objectives
- Count freight departures by weekday
- Identify busiest shipping days
- Visualize weekly freight activity trends

## Tools Used
- SQL Server
- Tableau
- Excel

## Files Included
- SQL queries
- Tableau dashboard
- Dataset
- Screenshots

## Key Insights
- Freight departures vary depending on the weekday
- Certain weekdays show higher transportation activity
- Tableau visualizations highlight weekly shipping trends clearly

## Skills Demonstrated
- COUNT()
- GROUP BY
- Date functions
- Trend analysis
- Data visualization
