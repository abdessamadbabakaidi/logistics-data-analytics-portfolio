Welcome to my logistics and freight analytics portfolio.

# End-to-End Analytics Workflow

This portfolio showcases complete end-to-end data analytics projects using a custom-built logistics dataset that was personally created and not downloaded from public internet sources.

Each project demonstrates:
- Data organization
- SQL querying
- Aggregation and business analysis
- Tableau dashboard development
- Visualization and reporting
- Portfolio documentation

The goal of these projects is to simulate real-world freight and transportation analytics workflows.
