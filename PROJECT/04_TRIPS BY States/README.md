# 04 - Trips By States

## Project Overview
This project analyzes freight trips across different states using SQL and Tableau.

## Objectives
- Count trips by state
- Identify states with the highest freight activity
- Visualize trip distribution geographically

## Tools Used
- SQL Server
- Tableau
- Excel

## Files Included
- SQL queries
- Tableau dashboard
- Dataset
- Screenshots

## Key Insights
- Some states generate significantly more freight trips
- Geographic analysis helps understand transportation activity
- Tableau dashboards provide clear state-by-state comparisons

## Skills Demonstrated
- COUNT()
- GROUP BY
- ORDER BY
- Geographic analysis
- Data visualization
