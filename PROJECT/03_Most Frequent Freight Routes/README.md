# 03 - Most Frequent Freight Routes

## Project Overview
This project analyzes the most frequently used freight routes using SQL and Tableau.

## Objectives
- Identify the most common freight routes
- Measure route frequency
- Visualize transportation patterns

## Tools Used
- SQL Server
- Tableau
- Excel

## Files Included
- SQL queries
- Tableau dashboard
- Dataset
- Screenshots

## Key Insights
- Certain freight routes are used significantly more often
- Route frequency helps identify major transportation corridors
- Tableau visualizations make route comparisons easier

## Skills Demonstrated
- COUNT()
- GROUP BY
- ORDER BY
- Business analysis
- Data visualization
