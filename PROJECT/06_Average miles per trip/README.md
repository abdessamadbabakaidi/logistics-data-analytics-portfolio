# 06 - Average Miles Per Trip

## Project Overview
This project analyzes the average distance traveled per freight trip using SQL and Tableau.

## Objectives
- Calculate average miles per trip
- Compare trip distances
- Visualize transportation distance patterns

## Tools Used
- SQL Server
- Tableau
- Excel

## Files Included
- SQL queries
- Tableau dashboard
- Dataset
- Screenshots

## Key Insights
- Average trip distance varies across freight operations
- Distance analysis helps understand transportation efficiency
- Tableau dashboards provide clear visualization of trip patterns

## Skills Demonstrated
- AVG()
- GROUP BY
- Aggregation analysis
- Data visualization
- Business analysis
