# 01 - Miles Per Carrier

## Project Overview
This project analyzes total miles completed by each freight carrier using SQL and Tableau.

## Objectives
- Calculate total miles per carrier
- Identify top performing carriers
- Visualize carrier performance

## Tools Used
- SQL Server
- Tableau

## Files Included
- SQL queries
- Tableau dashboard
- Screenshots
- Dataset

## Key Insights
- Some carriers completed significantly more miles
- Mileage distribution varies across carriers
- Tableau dashboard provides visual comparison

## Skills Demonstrated
- GROUP BY
- SUM()
- ORDER BY
- Data visualization
